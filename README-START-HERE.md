# Leggi questo per primo

Questo repo è **LifeOS**: una life dashboard personale, guest-first e
local-first — nove moduli sotto la shell `(app)`, dati in IndexedDB sul
dispositivo, sync opzionale con Supabase quando crei un account.

Il vecchio testo di questo file descriveva il progetto PRIMA del
rebuild (dashboard mock, task adattivi, health/finance log): quel mondo
è stato ritirato nei run 05-06 e questo file è stato riscritto al
run-09 perché non mentisse più.

- **Convenzioni per gli agenti**: [AGENTS.md](AGENTS.md)
